The Aseba user manual comprises the following introductory topics:

  • Getting started, a step-by-step introduction on how to program with Aseba, using the robot simulator challenge,
  • Studio, a description of Studio, the Aseba integrated development environment,
  • Language, a detailed description of the Aseba programming language, with examples.

And these advanced topics:

  • Remap Identifiers using asebaswitch, a way to put several single-node robots together in an aseba network.
